Little Twitter

Self-made to Practice React

Link to the App

This App is developed by myself using React.

I utlized Twitter as a template and Youtube videos as references to both study React.js and put what I have learned into practice.

The App included a sign up and log in system, in which I utlized JSON web token to encode the passwords of all users.

The functionality of the app is the same as the basic functions of Twitter.

User can post on the web, like & comment posts from other users, and edit & delete their own posts and comments.

I also built a backend from scratch using mongoDB.
