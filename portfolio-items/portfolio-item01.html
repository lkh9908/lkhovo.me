With three other teammates, in less than five days, we developed a social app from scratch using JavaScript, HTML, CSS, and a REST API for the backend.

app is fully deployed and can be accessed from the web anywhere.

To help people make new friends, Miaomiao utilizes a personality test and matches users base on the score they got.

Miaomiao also have a live chat system, which allows all users who are currently online to chat with each other.

At the core of our design is the anonymous post box, which allows people to ask their friends things they would not dare to ask in person!

If the user ges bored, there is an quiz game for them to relax.

Lastly, Miaomiao has a bad word filter system which forbid people to use bad words in their name or post-box related inputs.
